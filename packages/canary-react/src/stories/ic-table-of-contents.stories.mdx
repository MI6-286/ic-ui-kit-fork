# [Canary]: Table of contents
The table of contents component is used to create a navigable list of sections within a page or document. It can be positioned on the left, right or on top of the child content.

#### Basic code example

```jsx
import * as React from "react";
import { IcTableOfContents } from '@ukic/canary-react';

const IcTableOfContents = () => (
   <IcTableOfContents position="right" truncate={true}>
    <h1>Table of Contents</h1>
   </IcTableOfContents>
);

export default IcTableOfContents;
```
